Tic Tac Toe vs JavaScript

Click Here!

 Tic Tac Toe vs JavaScript is a fun game created with the intention of playing against an "AI". Javascript was the main language used. To generate the "AI's" moves the Math.random and Math.floor functions were used to generate random indexes on the grid. To ensure JavaScript could only play after the user had clicked a box and could only play unclicked boxes if/else statements were used. The grid boxes were created by using nested for-loops and an event listener was used to track the users moves. The goal was to create a simple Tic Tac Toe game that kept score and signaled the winner.
